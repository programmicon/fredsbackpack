Everything you need to get started as a Front-end Developer (a FRED), packed neatly into one package.

Download for Windows or Mac Download for Mac or Windows

What's inside?

  • Atom — a text editor to write your code
  • FileZilla — FTP solution to push your local site up to a live server
  • MAMP — easy local server environment
  • GitHub for Mac — GUI for using GitHub
  • GitHub for Windows — GUI for using GitHub
  • FRED's First Website — sample site to get you started
